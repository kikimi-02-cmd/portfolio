# portfolio

個人開発プロジェクトのポートフォリオサイト。

## 開発

```bash
npm install
npm run dev
```

http://localhost:3000 を開く。

## 構成

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4

## 編集ポイント

- `src/data/projects.ts` — 掲載プロジェクト
- `src/components/Hero.tsx` — トップの自己紹介
- `src/app/layout.tsx` — メタデータ
